# Automatización de Documentos en Word  

Este proyecto automatiza la generación de documentos en formato Word a partir de una plantilla y datos provenientes de un archivo CSV. Utiliza la biblioteca `docxtpl` para rellenar plantillas con información personalizada y `pandas` para manejar los datos.  

## 📌 Características  
- ✅ Carga de datos desde un archivo CSV.  
- ✅ Uso de plantillas en Word (`.docx`).  
- ✅ Generación masiva de documentos personalizados.  
- ✅ Organización automática de los archivos generados en un directorio específico.  

## 🛠 Tecnologías utilizadas  
- Python  
- `docxtpl`  
- `pandas`  
- `datetime`  
- `os`  

## 🚀 Cómo usar  
1. Ubica los archivos necesarios en la carpeta `Proyecto_02_AutomatizarWord/`.  
2. Asegúrate de que el archivo CSV contiene los datos requeridos.  
3. Ejecuta el script y los documentos generados se almacenarán en `documentos_generados/`.  

## 📂 Estructura del Proyecto  
```
Proyecto_02_AutomatizarWord/
│── sp-plantilla-rrhh-info.docx
│── sp_fake_data.csv
│── script.py
│── documentos_generados/  # Carpeta donde se guardarán los documentos generados
```
